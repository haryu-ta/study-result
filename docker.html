docker起動

  sudo systemctl start docker

イメージコマンド

コマンド 概要 使い方
docker image pull イメージファイルのダウンロード docker image pull {イメージ名}:[タグ名]/td>
docker image ls イメージの一覧表示
docker image rm イメージの削除 docker image rm [オプション] {イメージ名}
-f : 強制削除
docker image prune 未使用イメージの削除 docker image prune [オプション]
docker search イメージ検索 docker search [オプション] {検索キーワード}
docker login Docker Hubへのログイン docker login [オプション] [サーバ]
docker logout Docker Hubへのログイン docker logout
docker image push イメージのアップロード docker image push {イメージ名}:[タグ名]
docker image import tarファイルからイメージ作成 docker image import {ファイル or URL} {イメージ名}
docker image load tarファイルからイメージを生成 docker image load [オプション]

コンテナコマンド

コマンド 概要 使い方
docker container create コンテナの作成（起動なし） docker container create [オプション] {イメージ名}
--name : コンテナ名
docker container run コンテナの作成/起動 docker container run [オプション] {イメージ名}
オプション
docker container ls 稼働コンテナの一覧確認 docker container ls [オプション]
docker container stats コンテナの稼働確認 docker container ls {コンテナ識別子}
docker container start 停止コンテナの起動 docker container start {コンテナ識別子}
docker container stop コンテナの停止 docker container stop [オプション] {コンテナ識別子}
docker container restart コンテナの再起動 docker container restart [オプション] {コンテナ識別子}
docker container rm コンテナの削除 docker container rm [オプション] {コンテナ識別子}
docker container exec 稼動コンテナでプロセス実行 docker container exec [オプション] {コンテナ識別子} {実行するコマンド}
docker container top 稼動コンテナのプロセス確認 docker container top {コンテナ識別子}
docker container rename 稼動コンテナ名のリネーム docker container rename {旧プロセス名}　{新プロセス名}
docker container cp 稼動コンテナとのファイルコピー docker container cp {ホストファイル}　{コンテナ識別子}:{コンテナ内のファイルパス}
docker container cp {コンテナ識別子}:{コンテナ内のファイルパス}　{ホストディレクトリパス}
docker container commit コンテナからイメージ作成 docker container commit [オプション] {コンテナ識別子} {イメージ名}
docker container export コンテナをtarファイルに出力 docker container export {コンテナ識別子} > {tarファイル名}

docker container run --it --name testcontainer ubuntu /bin/bash


ネットワーク用コマンド

Dockerコンテナ間の通信を行いたいときに利用する。

コマンド 概要 使い方
docker network create ネットワークの作成 docker network create [オプション] {ネットワーク}
docker network ls ネットワークの確認 docker network ls
docker network connect ネットワークの接続 docker network connect [オプション] {ネットワーク} {コンテナ}
docker network disconnect ネットワークの接続解除 docker network disconnect
docker network rm ネットワークの削除 docker network rm [オプション] {ネットワーク}
