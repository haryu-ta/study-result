コレクション

            ●Map
            ①HashMap
            キーの並び順を保持しない
            高速

            ②TreeMap
            キーが自然順序、もしくはコンストラクタに指定されたComparatorに従って昇順にソート

            ③LinkedHashMap
            キーが挿入される挿入順

            ●List
            ①ArrayList
            ランダムアクセスをサポート
            要素の検索において高速に動作
            要素の追加、削除においては追加、削除された要素以降の要素を再配置する必要があるため低速

            ②LinkedList
            要素の追加、削除を行う場合、リンク情報を変更することによって処理するため高速に動作
            要素の検索においては、要素に順番にアクセスするため低速

            ●Set
            ①HashSet
            保持する要素内において重複要素を持ちません
            Setインタフェースを実装したクラス内では最も高速に動作します

            ②TreeSet
            保持する要素内において重複要素を持ちません
            保持する要素は要素の自然順序、もしくはコンストラクタに指定されたComparatorに従って昇順にソートされます

            ③LinkedHashSet
            保持する要素内において重複要素を持ちません
            要素の挿入される挿入順を保持します
