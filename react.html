??演算子 

data ?? value

→　dataが null または undefined の場合にvalueを返す
   それ以外はdataを返す
||演算子 

data || value

→　dataが null または undefined または 空文字 または 0 または false の場合にvalueを返す
   それ以外はdataを返す
